# useLinkedList

Just like `useState`, you can pass a default value (e.g. `{ data: 1, next: null }`)

```js
function App() {
  const [head, handlers] = useLinkedList()

  return (
    <>
      <button onClick={() => handlers.add(handlers.size() + 1)}>
        Add to linked list
      </button>
      <hr />
      <LinkedListIterator head={head} />
    </>
  )
}
```
