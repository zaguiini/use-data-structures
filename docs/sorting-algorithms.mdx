# Sorting algorithms

All the sorting algorithms follow the same pattern. Here are the algorithms we implemented:

- `useBubbleSort`
- `useSelectionSort`

We are **NOT** changing your array, we are creating a shallow copy of it.

### Usage

You can pass a custom predicate as the second argument to the hook to change how the array will be sorted.

Example: `(a, b) => a > b` to sort the list in descending order.

```js
import { useBubbleSort } from 'use-data-structures'

function App() {
  const unsortedArray = [64, 25, 12, 22, 11]
  const sortedArray = useBubbleSort(unsortedArray)

  return (
    <div>
      <h2>Unsorted array</h2>
      <p>{unsortedArray.join(', ').toString()}</p>

      <h2>Sorted array</h2>
      <p>{sortedArray.join(', ').toString()}</p>
    </div>
  )
}
```
